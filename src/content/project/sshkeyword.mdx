## Résumé du projet

{frontmatter.name} est un programme Python conçu pour rapidement se conencter à un serveur SSH à partir d'un terminal. Il permet aux utilisateurs de se connecter rapidement en utilisant des mots-clés préalablement définis. Le programme offre la possibilité d'ajouter plusieurs mots-clés et de configurer une connexion SSH par défaut.

Toutes les interactions et configurations avec ce programme se font via l'interface en ligne de commande (CLI), assurant une utilisation simple et rapide.

Je l'utilise tous les jours depuis plusieurs années
