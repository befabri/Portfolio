## Résumé du projet

{frontmatter.name} est un programme Python CLI conçu pour pouvoir rapidement se connecter en SSH à partir d'un terminal.
Il permet donc aux utilisateurs de se connecter en utilisant des mots-clés qui ont été préalablement définis. Il y a aussi la possibilité de configurer une connexion SSH par défaut afin de se connecter sans mot-clef.
