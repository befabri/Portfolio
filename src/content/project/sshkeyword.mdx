## Résumé du projet

{frontmatter.name} est un programme Python conçu pour pouvoir rapidement se connecter en SSH à partir d'un terminal. Je l'utilise pratiquement tous les jours depuis déjà plusieurs années. Il permet aux utilisateurs de se connecter rapidement en utilisant des mots-clés préalablement définis,
il y aussi la possibilité de configurer une connexion SSH par défaut.
Toutes interactions et configurations se font en ligne de commande.
