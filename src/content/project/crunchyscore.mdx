## Description du projet

L'extension ajoute les scores de MyAnimeList sous chaque anime ainsi que sur la page du lecteur vidéo de Crunchyroll. Les scores sont récupérés depuis une API développée en Fastify Node.js qui est connectée à l'API de MyAnimeList.

L'utilisateur a la possibilité de personnaliser son affichage avec diverses options, à savoir :

-   L'ordre de tri des animés
-   La couleur du texte de la note
-   L'étiquette du score
-   La précision des décimales

## Développé avec

-   **Frontend:** AstroJs, JavaScript, Vite, Tailwind CSS
-   **Backend:** Node.js, Fastify, TypeScript, Prisma, MySQL
-   **Extension:** JavaScript, HTML5, CSS3
